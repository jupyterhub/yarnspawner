yarnspawner
===========

A custom Spawner_ for JupyterHub_ that launches notebook servers on Apache
Hadoop/YARN clusters.


Installation
------------

**Install with Pip:**

.. code::

    pip install jupyterhub-yarnspawner

**Install from source:**

.. code::

    git clone https://github.com/jcrist/yarnspawner.git
    cd yarnspawner
    pip install .
